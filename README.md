### âš™ï¸ Setup Instructions

1. **Clone the Repository**

   ```bash
   git clone https://github.com/DurgaBhavani25/FindHelper.git
   cd FindHelper
   ```

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Run JSON Server**

   ```bash
   npm start
   ```
